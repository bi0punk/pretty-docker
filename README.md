# pretty-docker

Terminal User Interface (TUI) tool built with Textual that displays Docker container information in a clean, formatted way.

## Stack

Python 3, Textual, Docker SDK, Rich, PrettyTable

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
# TUI mode
python subprocess_docker.py

# Simple CLI output
python app.py
```
